mac使用笔记

  • mac
  • tips

Mac使用小技巧。

约定：命令行需在终端里运行

1.选中一个文件按 enter 可以直接改名的.

cmd + up 是回到上一层文件夹
cmd + down 如果是文件夹就进入文件夹，如果是文件就打开（比如说 .avi 就会自动调用播放器）

2.OS X 中空格键可以预览 几乎所有的 东西

3.查看多个文件占用的容量

ctrl + command + i

4.把多个文件归类到一个文件夹中：选中你想要的文件

control＋command＋n

5.快速关机 正常左上角苹果－关机会出来个对话框 按住option再点关机就不会出现了 其实按住option很多菜单会发生些小变化，自己探索吧～

6.调整声音的时候，系统会发出“嘟嘟”的声音，按住shift键就没了

7.同时按shift+option， 可以4分1格调节（这个同样适用键盘背光和屏幕亮度）

8.剪切

ctrl+c (复制)
cmd+alt+v (剪切)

9.显示隐藏文件

defaults write com.apple.finder AppleShowAllFiles -boolean true ; killall Finder

恢复

defaults write com.apple.finder AppleShowAllFiles -boolean false ; killall Finder

10.finder标题栏显示完整路径,且在小图标上右键可切换目录

defaults write com.apple.finder _FXShowPosixPathInTitle -bool YES

11.随时随地用emoji表情卖萌

control + command + space

12.删除光标后的文字

fn+delete

13.删除光标前方的单词（只对英文有效哦，是一个单词）

alt+delete

14.删除光标前的整个句子

command+delete

15 解决launch图标混乱，有些应用明明已经卸载了，但图标还留在launch上，输入下面的命令即可解决，注意：这也会将自定义的文件夹恢复成初始化

defaults write com.apple.dock ResetLaunchPad -bool true;killall Dock

16 安装Homebrew

ruby -e “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install

17 安装markdown

brew cask install macdown

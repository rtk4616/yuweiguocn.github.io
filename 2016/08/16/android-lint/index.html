Android Lint提示的错误及解决方案

  • android
  • lint
  • perfermance

状态：未完成

性能问题（Performance issues）

  • 手动复制数组到集合（Manual array to collection copy）

解决方案：

  • 冗余的’String.toString()’（Redundant ‘String.toString()’）
    解决方案：直接把toString去掉就可以了

  • 冗余的String构造方法调用（Redundant String constructor call）

  • 字符串连接作为’SpannableStringBuilder.append()’的参数调用(String concatenation as argument to ‘SpannableStringBuilder.append()’ call)

可能有问题（Probable bugs）

  • Call to ‘toString()’ on array

  • Mismatched query and update of StringBuilder

  • Mismatched read and write of array

  • Statement with empty body

  • statement has empty body

  • Throwable instance not thrown

  • Runtime exception instance ‘new RuntimeException(“mClazz is null”)’ is not thrown

  • Unused assignment

  • Variable is never used
  • Variable initializer is redundant

Verbose or redundant code constructs

  • Redundant type cast
